Bio Plant pH Prediction (Data Analyst Project)

This project analyzes sensor data from a bio-treatment facility to predict pH levels at different treatment stages using machine learning techniques.

🔍 Overview

Models used include Linear Regression and Neural Networks with various activation functions. Performance is evaluated using MSE and MAE, and insights are visualized using loss curves.

🛠️ Tools & Techniques

  • Python, pandas, scikit-learn, TensorFlow, matplotlib
  • Data cleaning, scaling, feature extraction
  • Model evaluation and performance benchmarking
